# Contributing to Yaademan

Thank you for helping us remember.

## How to Add a Memorial

1. Fork this repository
2. Create a new JSON file in `data/people/`
3. Follow the schema in `data/schema/person.schema.json`
4. Add photos to `images/people/[person-id]/`
5. Submit a pull request

## Data Guidelines

- All information must be verifiable
- Include sources when possible
- Be respectful and accurate
- Photos should be appropriate memorial images

## Review Process

All submissions are reviewed before being published.
